////////////////////////////////////////////////////////////////// 
// changing app name
//////////////////////////////////////////////////////////////////

First of all: 
	change displayName value in app.json
		{
    		"name": "SomethingSomething",
    		"displayName": "My New App Name"
		}
For Android app: 
	change string tag value in strings.xml file 
		<string name="app_name">My New App Name</string>
For iOS: 
	replace the value after <key>CFBundleDisplayName</key> in info.plist
		<key>CFBundleDisplayName</key>
		<string>My New App Name</string>

-then rebuild the application

// for android
- open android folder and copy all minmap folders
- paste them and replace folders in >>>> YourProject -> android -> app -> src -> main -> res
// for IOS
- open ios -> AppIcon.appiconset, then copy all the files
- paste them and replace folders in >>>> YourProject -> ios -> YourProject -> Images.xcassets -> AppIcon.appiconset
